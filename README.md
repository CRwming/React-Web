# React-WebApp

## 描述
- WebApp使用ReactJS、amazeui和material-ui使用Webapck打包工具，实现类似原生应用的功能；

- WebApp 使用了第三方提供的B站API接口；

## 使用

1. cd React-WebApp

2. npm install

3. npm run dev
